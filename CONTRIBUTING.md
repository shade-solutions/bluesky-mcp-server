# Contributing

Thanks for helping improve the project.

## Local Development

```bash
bun install
bun run typecheck
bun run build
bun run dev
```

## What to Include

- Keep changes small and focused.
- Update docs when behavior changes.
- Prefer open-source friendly defaults and clear config names.
- Validate that `bun run typecheck` and `bun run build` pass before opening a PR.

## Pull Requests

- Describe what changed and why.
- Include screenshots or examples when the docs change.
- Mention any new environment variables or setup steps.
